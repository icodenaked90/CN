TokoVOIP
TokoVOIP er vores nye lyd system, lavet med et custom TS plugin.

Det giver os mange flere fordele, som kan læses her:
  • Bedre lyd kvalitet
  • Mulighed for radioer
  • Lyd alt efter afstand
  • Telefonopkald

TokoVOIP er påkrævet installeret og skal køre for at spille på vores server.

Du kan finde information og download links på: https://ts.fivem.dk
